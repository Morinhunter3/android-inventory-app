Android Inventory Management App

Overview

This Android Inventory Management Application was developed as part of the Southern New Hampshire University Computer Science program.
The application allows users to manage inventory items through a mobile interface while demonstrating software engineering, database management, and algorithm implementation concepts.

Features

- User account creation and login
- Secure password storage using SHA-256 hashing
- Inventory item creation, editing, and deletion
- SQLite database integration
- Inventory search functionality
- Inventory filtering for low-stock and out-of-stock items
- Inventory sorting by name and quantity
- HashMap-based item lookup for improved efficiency
- SMS notification settings with preference persistence
- Input validation and error handling
- Logout functionality

Technology used

- Java
- Android Studio
- Android SDK
- SQLite
- RecyclerView
- SharedPreferences
